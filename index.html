JATO Veri Endpoint'leri

JATO veritabanından araç verilerini alan endpoint'ler. Bu endpoint'ler sadece Türkçe dil desteği sunar ve senkronizasyon yapmadan doğrudan veri çekimi yapar.

JATO Farklılıkları:
  • Veri Kaynağı: JATO veritabanı (Brisa backoffice değil)
  • Dil Desteği: Sadece tek dil (tr)
  • İşlem Tipi: Sadece veri çekimi (senkronizasyon yok)

Bridgestone
{brand}-lastikleri-{groupType}
Örnek: https://www.bridgestone.com.tr/alfa-romeo-lastikleri-binek
KULLANILMIYOR
{brand}-{groupType}lastikleri
Örnek: https://www.bridgestone.com.tr/toyota-bineklastikleri

Path Arama (2.3M+ Kayıt)

TyreCategoryLink veritabanında arama yapın. Marka, model, yıl ve diğer filtrelere göre URL path'lerini bulun.

Sunucu Gerekli: 2.3 milyon kayıt tarayıcıda işlenemez. Arama için Flask API sunucusunu çalıştırın: 
pip install -r requirements.txt
python create_database.py  # CSV'yi SQLite'a dönüştür
python api_server.py       # API sunucusunu başlat
